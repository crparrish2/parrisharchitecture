APARTMENTS + RETAIL BUILDINGWESTMINSTER WOODS
This is a proposal for a mixed-use building containing 6 apartments and 3,000 sq ft of retail space. The design problem was to create a building which expressed a form-based urban building type but would be suitable for a suburban location. In this case, this has been achieved by building close to the street, providing grade related retail uses with apartments above and locating the parking behind. Although the site is small and isolated within its suburban context it is possible that it will become an attractive node for the community. The style of the building draws inspiration from many types of traditional and contemporary lifestyle centers in North America.
Status: Concept Proposal 2010
Project Type: Residential / Commercial mixed-use
Design Field: Architecture / Urban Design
Building Size: 14,100 sq ft approx
Location: Guelph ON
